# GoImageLoader
A example load images on android

#实现功能
> * 多线程加载图片
> * 可设置加载图片顺序（任务开始执行的顺序，执行结果的顺序没有控制）
> * 图片的缓存（通用的两级缓存：内存和磁盘）
> * 避免图片的错位和闪屏（使用的设置tag方式）
> * 图片加载完成显示带动画效果（现在只实现了简单的渐变透明度的效果）
> * 优化图片显示效率（只加载当前界面内图片，使用对scroll监听方式实现），这个还没有实现
